loadFile(file_fn, fileType)

source code 

Determine how to load a file based on the extension & the fileType

Parameters:
  • file_fn (string) - the filename of the file to be loaded.
  • fileType (string) - the fileType to loaded.
Returns:
the loaded file

The following are valid fileTypes:

  1. 'cc'|'clustCoeff' is the clustering coefficient
  2. 'deg'|'degree' is the local vertex degree
  3. 'eig'|'eigen' is the eigenvalues
  4. 'mad'|'maxAvgDeg'
  5. 'ss1'| 'scanStat1'
  6. 'ss1'| 'scanStat1'
  7. 'tri'|'triangle'
