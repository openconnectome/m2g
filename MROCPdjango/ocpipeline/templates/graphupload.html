Please upload the graph(s) with largest connected components(LCC), and ROIs for each graph you would like invariants to be computed.

You must use a single zipfile (.zip) containing ALL data in a single directory even for multiple graphs.
Note the following file naming convention for single graph below where the base file name is the the same:

  • The base file name here is graph1
  • Graph file name: graph1_fiber.mat or graph1_bggr.mat are acceptible
  • LCC file name: graph1_concomp.mat
  • ROI's: graph1_roi.xml and graph1_roi.raw
