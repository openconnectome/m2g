Programmatic use:

To use a programmatic version please refer to our github sample code for examples of use and sample python scripts.
The base url for an upload is always http://mrbrain.cs.jhu.edu/disa/upload/ .
The example uploadsubj.py can be used as follows from a terminal session:
python uploadsubj.py http://mrbrain.cs.jhu.edu/disa/upload/{YOUR_project}/{YOUR_site}/{YOUR_subject}/{YOUR_session}/{scanID}/yes|no. The last param is either 'yes' or 'no', but not both: 'yes' adds .mat files for graph invariant result and .csv for graphs, while 'no' does not'.
Multi-subject/scan example is also provided for batch uploading and processing of multiple scans. The use is similar to uploadsubj.py.
