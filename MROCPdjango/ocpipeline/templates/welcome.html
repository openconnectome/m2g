Welcome to the Magnetic Resonance One-Click Pipeline (MROCP) for Connectomes [ALPHA]

Welcome! This is an Open Source Project that provides Web Services to perform a scientifically interesting tasks involving Connectomes.
Services here can either be accessed programmatically or via web forms. Want to learn more about the OpenConnectome projects?

Form based use

You may choose to:
  • Upload derivatives (fiber tractography and region of interest files) , build a graph and optionally compute invariants.
  • Compute Invariants of graphs previously built - including:
    • The Scan Statistic 1
    • The Triangle Count
    • The Clustering Coefficient
    • The Maximum Average Degree
    • The Local Degree Count
    • The Top 100 eigenvalues
    • The Scan Statistic 2 [Coming soon]
    • The Average Path Length [Coming soon]
    • The Graph Diameter [Coming soon]
  • Convert data files between npy, mat and csv formats. Data files include built graphs, largest connected component, and all invariants. Some conversion formats unavailable.

Programmatic use

All services can also be accessed programmatically as follows:

  • Upload derivatives to build a graph

    This service's base url for an upload is http://mrbrain.cs.jhu.edu/disa/upload/ .
    A single subject graph build example can be found here uploadsubj.py.
    Multi-subject/scan example is also provided for batch uploading and processing of multiple scans.
    Detailed use cases are available for more information on usage.

  • Compute Invariants of built graphs

    This can be accessed using the base url http://www.mrbrain.cs.jhu.edu/disa/graphupload/. An example of usage can be found here. Big graph testing incomplete.
    Detailed use cases are available for more information on usage.

  • Convert data file formats

    The use is similar to building a graph programmatically. The base url is http://mrbrain.cs.jhu.edu/disa/convert/.
    Single and batch conversion examples are available in our examples repository.
    Detailed use cases are available for more information on usage.
